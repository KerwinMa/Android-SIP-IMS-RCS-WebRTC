The Jibe App-to-App SDK provides developers with a simple, easy-to-use set of functions, which enable their applications to use realtime communication features, such as live audio, live video, low-latency data, chat and file sharing, with just a few lines of code. Communication is established directly via the mobile network operator's IMS/RCS infrastructure, which delivers the fastest possible mobile-data transmission, via peer-to-peer communication.

The Jibe SDK has been designed to make the best use of the programming paradigms available on each target platform. This enables developers to quickly integrate realtime features without having to learn new techniques or protocols.
